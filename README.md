# homeagent — Бармен

AI character-агент в терминале: бармен **Виктор**, который ведёт разговор, предлагает выпить, описывает приготовление коктейлей, оценивает степень опьянения по вашим репликам и в нужный момент твёрдо отправляет домой. Лицо бармена в TUI меняется в зависимости от его настроения.

- [docs/SPEC.md](./docs/SPEC.md) — спецификация
- [docs/PLAN.md](./docs/PLAN.md) — план реализации
- [docs/](./docs/) — все спецификации и планы

## Запуск

```bash
npm install
cp .env.example .env   # вписать API-ключ
npm run dev
```

## Провайдер

По умолчанию используется **OpenCode Go** (`https://opencode.ai/zen/go/v1`) с моделью `deepseek-v4-pro`. Также поддерживаются `anthropic` и `openai`.

Способы выбора провайдера (приоритет):
1. `BARTENDER_PROVIDER` в `.env` (поверх всего).
2. Сохранённый выбор в `~/.homeagent/preferences.json` (после первого ручного выбора).
3. **Экран выбора при старте** — появляется, если провайдер не выбран и настроено больше одного (или ни одного).
4. Единственный настроенный провайдер выбирается автоматически.

Сменить провайдер в рантайме: команда `/settings` → «Провайдер LLM». Текущая беседа сохраняется.

## Команды и управление

| Команда / клавиша | Действие |
|---------|----------|
| `/menu` | Меню коктейлей |
| `/settings` | Настройки: сменить провайдера, перезапустить вечер, помощь, выход |
| `/help` | Подсказка по командам |
| `/leave` | Попрощаться с барменом |
| `/exit` | Выйти (с подтверждением) |
| `/state` | Внутреннее состояние (debug) |
| `ESC` / `Ctrl+C` | Подтверждение выхода (ESC/Y/Enter — уйти, иная клавиша — остаться) |

Обычный текст — реплика бармену.

## Стек
